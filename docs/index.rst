Welcome to APRSD Location Plugin's documentation!
====================================================

The APRSD Location Plugin allows users to query the last known GPS location of any APRS callsign via APRS messages. It uses aprs.fi to retrieve GPS coordinates and geopy to reverse geocode them into human-readable locations.
